Welcome to RESTLess

RESTLess is a WPF REST Client for windows.

Design

Authors and Contributors

RESTLess is written by @leegould.

Source code

To get the source code from git:

$ git clone https://github.com/leegould/RESTLess.git

Building and running

The project was written using VS2013, and you should be able to build the main project and run as normal.

Installation

There is a setup project which builds to an MSI for installation.

Support or Contact

Having trouble? Create an issue or contact the author at https://github.com/leegould/RESTLess
